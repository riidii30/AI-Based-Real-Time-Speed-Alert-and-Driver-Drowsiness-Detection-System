# AI-Based Real-Time Speed Alert & Driver Drowsiness Detection System

## Features
- Real-time speed monitoring
- Driver drowsiness detection
- WhatsApp overspeed alerts
- AI voice alerts
- Live location tracking
- Trip summary notifications
- MySQL trip history storage
- Admin dashboard for analytics

## Tech Stack
- Python
- Flask
- JavaScript
- HTML/CSS
- MySQL
- OpenCV
- MediaPipe
- Twilio WhatsApp API

## Run Project

python3 run_project.py
